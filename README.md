# 🧰 Installation

## Prerequisites

- [`Python`](https://www.python.org/downloads/release/python-3913) 3.12.1
- [`Poetry`](https://python-poetry.org/docs/#installation) >=1.7.1

Install packages before continue

```bash
poetry install
```

# ⌨ Development

## 🧪 Running tests

```bash
poetry run python -m pytest
```
